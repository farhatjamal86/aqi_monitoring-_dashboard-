# India AQI Dashboard

An interactive Air Quality Monitoring Dashboard built using Python and Streamlit to analyze AQI trends, pollutant levels, health impact indicators, and geographic air-quality patterns across India.

## Project Overview

This project visualizes air quality data from multiple Indian cities and helps understand:

- AQI trends over time
- Pollutant concentration levels
- Seasonal air quality variations
- Health impact indicators
- Geographic AQI distribution across India

The dashboard is designed to make environmental data easier to explore and understand through interactive visualizations and maps.


## Features

- Interactive AQI trend analysis
- Pollutant comparison charts
- Health impact visualization
- India AQI map using geographic coordinates
- City-wise AQI monitoring
- Time-series analysis
- AQI category classification
- Dark-themed interactive dashboard UI


## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Folium
- Streamlit-Folium


## Dataset Information

The dataset contains:

- AQI values
- PM2.5, PM10, NO2, SO2, CO, O3, NH3 pollutant data
- City and state information
- Geographic coordinates
- Temperature and humidity
- Population and industrialization indicators
- Health impact metrics
- Monthly and yearly trends



## Dashboard Sections

### AQI Overview
Displays overall AQI trends and average AQI statistics.

### City AQI Analysis
Provides city-specific AQI monitoring and pollutant analysis.

### Pollutant Levels
Compares major pollutants including PM2.5, PM10, and NO2.

### Health Impact Analysis
Visualizes respiratory admissions and asthma-related indicators.

### India AQI Map
Interactive geographic visualization of AQI levels across Indian cities.
